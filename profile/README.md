**SolverForge** is a community-driven initiative to continue and expand upon the work started by [Timefold](https://www.timefold.ai)'s now-discontinued Python solver.

Our first objective is to provide a [first-class, open-source AI constraint solver for Python](https://github.com/SolverForge/solverforge-legacy) that is able to provide optimal solutions to problems such as Vehicle Routing,
Employee Rostering, Maintenance Scheduling and more.

Our longer term objective is providing language independence through a universal, [native wrapper and type translator for the Timefold JVM](https://github.com/SolverForge/solverforge) that is compatible with bindings for any language.
